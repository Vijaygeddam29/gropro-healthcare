4. Financial MRI Engine (Core Product)
This combines all insights into one score.

Example logic:

class FinancialMRIEngine:

    def calculate_score(self, tax_score, pension_score, structure_score):

        score = (tax_score + pension_score + structure_score) / 3

        return round(score,2)

5. Pension Strategy Agent
Handles complex issues like:

NHS pension

tapered allowance

annual allowance

Example code:

class PensionAnalysisAgent:

    def analyse(self, income, pension_contribution):

        allowance = 60000

        if income > 260000:
            allowance = 10000

        excess = pension_contribution - allowance

        return {
            "allowance": allowance,
            "excess": excess
        }

10. Guardrails
Every AI agent must follow rules:

AI cannot give financial advice
AI must suggest scenarios

class ComplianceGuardrail:

    def check(self, recommendation):

        if "invest now" in recommendation:
            return "Review with advisor"

        return recommendation
